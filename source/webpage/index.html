Goals of the Project

Create a web application that acts like a scoreboard for competitive games. The goal of the project is to become more familiar with web application development using a JavaScript front-end framework, back-end framework, as well as either a relational or NoSQL database. Goals include implementing Discord integration for logging in to the application and displaying profiles. Other goals include setting up parts or the whole project in a Docker container, allowing project members to easily set up their environment. An initial goal would be to have the database as a container.
